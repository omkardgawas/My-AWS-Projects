# 🚀 EC2 Web Server Deployment using Nginx

## 📌 Project Overview

This project demonstrates how to launch an EC2 instance and deploy a static website using Nginx.

---

## 🛠️ Services Used

* AWS EC2
* Security Groups
* SSH
* Nginx (Web Server)

---

## 🚀 Steps Performed

1. Launched EC2 instance (Ubuntu)
2. Configured security group (SSH + HTTP)
3. Connected using SSH
4. Installed Nginx
5. Deployed website files
6. Accessed website via public IP

---
