# 🌐 Static Website Hosting using AWS S3

## 📌 Project Overview

This project demonstrates how to host a static website using AWS S3 without using any server.

The website is publicly accessible via an S3 website endpoint.

---

## 🛠️ Services Used

* AWS S3 (Simple Storage Service)
* IAM (for access control)

---

## 🚀 Steps Performed

1. Created an S3 bucket with a unique name
2. Disabled "Block all public access"
3. Uploaded static website files (HTML/CSS)
4. Enabled static website hosting
5. Configured bucket policy for public access
6. Accessed the website using the S3 endpoint URL
